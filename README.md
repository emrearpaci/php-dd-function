# Php dd function
Dump and die

## Sample
```
$example = 'Hello World!';
dd($example);

Result : 

function.php
---------------------------------
string 'Hello World!' (length=12)
```
---------------------------------
```
$array = ['Hello World!' , 'Hello Turkey!'];
dd($array);

Result : 

function.php
---------------------------------
array (size=2)
  0 => string 'Hello World!' (length=12)
  1 => string 'Hello Turkey!' (length=13)
```
